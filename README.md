# 🌐 SocialConnect

## 📖 Sobre o projeto

O SocialConnect é um projeto desenvolvido com o objetivo de simular uma plataforma de conexão entre pessoas, permitindo a apresentação de informações como equipe e formas de contato. Este projeto foi criado para praticar o uso de Git e GitHub, incluindo versionamento, trabalho em equipe e organização de código.

## 🚀 Tecnologias utilizadas

* HTML
* CSS
* Git
* GitHub
